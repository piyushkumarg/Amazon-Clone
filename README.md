# Amazon-Clone
This is a clone of popular e-commmerce website Amazon. I have created it with HTML CSS and React in frontend, Node and Express for backend and MongoDB for storing data.

NPM Packages Used
React-redux
Material UI
Express
Nodemon
Mongoose
